# Расчет уклона трубопровода (TUKLON)

Автоматический LISP-скрипт для AutoCAD (и аналогов), предназначенный для быстрого вычисления уклонов между двумя высотными точками и автоматической записи вычисленного значения в выбранное текстовое поле чертежа. 

Идеально подходит для проектировщиков наружных и внутренних инженерных сетей (НВК, ТС, ГСВ), генпланистов и геодезистов, выполняющих исполнительные съемки самотечных и напорных трубопроводов.

---

## Основные возможности

* **Интеллектуальное определение расстояния:** Длину трассы (знаменатель уклона) можно получить двумя способами:
  1. Кликнув по графическому элементу (Линия, Полилиния, Дуга или Сплайн) — скрипт сам считает его точную длину.
  2. Кликнув по уже существующему на чертеже тексту с расстоянием.
* **Автоматическое форматирование:** Результат уклона автоматически округляется до 4 знаков после запятой (например, `0.0055`), что соответствует стандартам оформления продольных профилей и планов сетей.
* **Циклический режим работы:** Команда работает непрерывно. Вы можете последовательно вычислять уклоны для целой сети колодцев или участков трубопровода без необходимости повторного вызова команды.

---

## Инструкция по установке и запуску

1. **Загрузка скрипта в AutoCAD:**
   * Введите в командной строке `APPLOAD` (или `ЗАГРИСП`).
   * Выберите файл `Tuklon.lsp` и нажмите **Загрузить** (Load).
   * При желании добавьте его в «Автозагрузку» (Startup Suite), чтобы скрипт подгружался автоматически при каждом открытии программы.

2. **Запуск команды:**
   * Введите в командную строку: `TUKLON`

3. **Порядок работы:**
   * **Шаг 1:** Выберите текст первой высотной отметки (например, лоток верхнего колодца `145.23`).
   * **Шаг 2:** Выберите текст второй высотной отметки (например, лоток нижнего колодца `144.11`).
   * **Шаг 3:** Укажите объект расстояния (кликните на текст с длиной участка).
   * **Шаг 4:** Кликните на текст, куда нужно записать итоговый уклон (например, на проектную надпись над линией `0.0000`).
   * **Результат:** Текст обновится на рассчитанный уклон, а в командную строку AutoCAD выведется подробный лог вычислений (перепад, длина, отметки).
