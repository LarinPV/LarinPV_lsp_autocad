# PROF.lsp — Автоматическое построение продольного профиля трассы

Скрипт на AutoLISP для AutoCAD, предназначенный для автоматизированного построения продольного профиля инженерных коммуникаций по плану трассы. Скрипт генерирует линии земли и трубы, расставляет пересечки, а также формирует готовую подпрофильную таблицу с расчетом уклонов, длин и глубин заложения.

**Автор:** LarinPV
**Год:** 2026

---

## ?? Возможности

* **Построение профиля:** Автоматическая отрисовка линии поверхности земли и верха трубы с заданными масштабами (Горизонтальный — 1:500, Вертикальный — 1:100).
* **Генерация подпрофильной таблицы:** Автоматический расчет и заполнение строк:
  * Уклон / Длина
  * Отметки земли
  * Отметки верха трубы
  * Глубина заложения
  * Пикет (с шагом 100 м)
  * Расстояния между точками
* **Умное распознавание пересечек:** При клике по примитивам (линии/полилинии) на плане скрипт автоматически определяет тип пересечения на основе слоя или типа линии (пересечения с дорогами, кабелями связи, электрокабелями, теплотрассами, газопроводами, канализацией и водопроводом).
* **Интерполяция данных:** Автоматическое вычисление пропущенных отметок земли и трубы между заданными точками.
* **Защита от ошибок (Анализ аномалий):** Выявление резких перепадов отметок земли (? 20 м при нулевой отметке) с автоматической пометкой «перепроверить отметку».
* **Учет дополнительных элементов:** Распознавание углов поворота (текст, начинающийся с «УП») и автоматическая отрисовка крановых узлов.
* **Автоматическая структуризация слоев:** Создание и распределение элементов по специализированным слоям.

## ?? Установка и запуск

1. Откройте чертеж в AutoCAD.
2. Вызовите команду `_APPLOAD` (или `ЗАГРПРИЛ` в русской версии).
3. Выберите файл `PROF.lsp` и нажмите «Загрузить».
4. В командной строке введите команду для запуска:

   PROF
