# Расчет глубины заложения трубопровода (TGLUB)

Удобный LISP-скрипт для AutoCAD (и аналогичных CAD-программ), автоматизирующий вычисление глубины заложения инженерных сетей. Скрипт в реальном времени рассчитывает разницу между отметкой поверхности земли и проектной отметкой лотка/верха трубы, после чего автоматически записывает результат в выбранный текст на чертеже.

Инструмент значительно ускоряет камеральную обработку профилей, построение сечений и оформление исполнительных или проектных планов сетей НВК (водопровод, канализация), теплосетей и газопроводов.

---

## Основные возможности

* **Прямое математическое вычисление:** Избавляет от необходимости считать разность отметок на калькуляторе и вручную перебивать текст.
* **Гарантированное форматирование:** Скрипт содержит встроенный алгоритм жесткого контроля разрядов, который гарантирует вывод строго двух знаков после запятой (например, `1.50` или `2.05`), что полностью соответствует требованиям ГОСТ к оформлению рабочих чертежей и схем.
* **Циклический режим работы:** После выполнения одного расчета команда не прерывается. Вы можете быстро «прокликать» и проставить глубины на протяжении всей кабельной или трубной трассы.

---

## Инструкция по установке и запуску

1. **Загрузка скрипта в AutoCAD:**
   * Введите в командной строке `APPLOAD` (или `ЗАГРИСП`).
   * Выберите файл `Tglub.lsp` и нажмите **Загрузить** (Load).
   * Добавьте файл в «Автозагрузку» (Startup Suite), чтобы утилита была доступна при каждом запуске программы.

2. **Запуск команды:**
   * Введите в командную строку: `TGLUB`

3. **Порядок работы:**
   * **Шаг 1:** Выберите текстовый объект с отметкой земли (например, `154.20`).
   * **Шаг 2:** Выберите текстовый объект с отметкой трубы (например, лоток канализации `152.10`).
   * **Шаг 3:** Выберите целевой текст на чертеже, куда необходимо записать результат (например, шаблон условного обозначения или старое значение глубины).
   * **Результат:** Целевой текст мгновенно изменится на точное вычисленное значение (в данном примере — `2.10`). Скрипт сразу предложит выбрать следующую отметку земли для непрерывной работы. Для выхода из команды нажмите **Esc**.
