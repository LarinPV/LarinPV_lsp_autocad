# Сетка перекрестий (Геодезическая сетка крестов)

Автоматический LISP-скрипт для AutoCAD (и аналогов), который вычисляет габариты выбранных объектов и строит вокруг них правильную координатную сетку перекрестий в соответствии с заданным масштабом.

---

## Основные возможности

* **Полная автоматизация блоков:** Больше не нужно предварительно искать или создавать блоки крестиков в чертеже. Скрипт сам генерирует блок нужного размера «на лету».
* **Универсальные масштабы:** Поддерживает любые геодезические масштабы. Автоматически рассчитывает пропорции:
  * **1:500** — Шаг `50м`, длина креста `3м` (блок `500_012`)
  * **1:1000** — Шаг `100м`, длина креста `6м` (блок `1000_012`)
  * **1:2000** — Шаг `200м`, длина креста `12м` (блок `2000_012`)
* **Умное округление:** Координаты крестов строго кратны шагу сетки. Алгоритм корректно работает как в положительной, так и в отрицательной зоне координат (ничего не смещается).
* **ГОСТ-овский цвет:** Элементы внутри блока создаются на слое `0` с фиксацией цвета индексного номера **92** (спокойный серо-зеленый оттенок, стандартный для топопланов).
* **Высокая скорость:** Благодаря использованию метода `entmake` вместо стандартной команды вставки, сетка из сотен крестов вычерчивается мгновенно, без мусора в командной строке и сбоев из-за включенных объектных привязок (OSNAP).

---

## Инструкция по установке и запуску

1. **Загрузка скрипта в AutoCAD:**
   * Введите в командной строке `APPLOAD` (или `ЗАГРИСП`).
   * Выберите файл `Сетка перекрестий.lsp` и нажмите **Загрузить** (Load).
   * *(Опционально)* Добавьте его в «Приложения» (Startup Suite), чтобы скрипт загружался автоматически при каждом открытии AutoCAD.

2. **Запуск команды:**
   * Введите в командную строку: `Сетка_перекрестий`

3. **Порядок работы:**
   * **Шаг 1:** Скрипт запросит масштаб: `Введите масштаб (например: 500, 1000, 2000) <500>:`. Введите число или просто нажмите *Enter* для выбора 500-го масштаба.
   * **Шаг 2:** Выберите рамкой объекты на чертеже, вокруг которых необходимо построить сетку. Нажмите *Enter*.
   * **Результат:** В чертеже автоматически создается слой `Геопункты` (если его не было), генерируется нужный блок и расставляется сетка с запасом по краям.

## Техническая информация для разработчика

Если захочется изменить логику работы «под себя», обратите внимание на эти переменные в теле кода:
* `step` — отвечает за шаг сетки (вычисляется как $Масштаб / 10$).
* `offset` — отступ («запас») сетки от крайних точек выбранных объектов. По умолчанию равен `(* 1.5 step)`.
* `half_len` — половина длины перекрестия от центра. Задает итоговый размер креста.
* `(62 . 92)` — DXF-код, отвечающий за цвет линий крестика. `92` можно поменять на любой другой номер цвета AutoCAD.
