# Расчет отметки газопровода с допуском (TGgaza)

Специализированный LISP-скрипт для AutoCAD (и аналогов), разработанный для автоматизации подготовки исполнительных чертежей и продольных профилей газопроводов. Скрипт считывает отметку земли с префиксом `з.`, вычитает из нее заданную глубину заложения и автоматически генерирует реалистичный случайный допуск (в пределах от `0` до `5 см`), имитирующий естественную неровность укладки трубы в траншею.

Новая расчетная отметка с префиксом `г.` автоматически создается на чертеже строкой ниже исходного текста.

---

## Основные возможности

* **Генерация случайного отклонения:** Функция использует процессорные такты `CPUTICKS` для создания неповторяющегося случайного сдвига от `0.00` до `0.05 м`. Это позволяет быстро формировать «живую» исполнительную документацию, где отметки незначительно колеблются, проходя все проверки технадзора.
* **Автоматическое позиционирование:** Новый текст создается ровно под исходным текстом земли. Шаг смещения автоматически рассчитывается на основе текущей системной переменной высоты шрифта `TEXTSIZE`.
* **Наследование свойств:** Создаваемый текст отметки трубы автоматически принимает слой, стиль, высоту и цвет исходного текста земли, сохраняя идеальное единообразие оформления чертежа.

---

## Инструкция по установке и запуску

1. **Загрузка скрипта в AutoCAD:**
   * Введите в командной строке `APPLOAD` (или `ЗАГРИСП`).
   * Выберите файл `Tggaza.lsp` и нажмите **Загрузить** (Load).
   * Добавьте файл в «Автозагрузку» (Startup Suite) для постоянного доступа.

2. **Запуск команды:**
   * Введите в командную строку: `TGgaza`

3. **Порядок работы:**
   * **Шаг 1:** Скрипт попросит ввести базовую глубину заложения: `Введите глубину (например 1.91):`. Введите число и нажмите *Enter*.
   * **Шаг 2:** Укажите на чертеже текст отметки земли, содержащий префикс `з.` (например, `з.146.55`).
   * **Результат:** Скрипт извлечет число `146.55`, вычтет из него `1.91` и добавит случайный миллиметровый допуск (например, `0.012`). Строкой ниже автоматически создастся новый текст: `г.144.63`.

---
## Техническая информация для разработчика

* По умолчанию случайный допуск генерируется формулой `(* (rem (getvar "CPUTICKS") 100) 0.0005)`. Максимальный порог — `0.05м`. Если вам нужно изменить величину колебания (например, до `2 см`), замените коэффициент `0.0005` на `0.0002`.
* Расстояние между строками земли и трубы задается множителем `(* (getvar "TEXTSIZE") 1.5)`.
